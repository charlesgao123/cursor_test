### 9.Inline Edit
#### 9.1 Quick Edit(Edit and Generate): Ctrl + K
选中代码，表示修改此处代码(Edit)。如：帮我提供这个函数的另外一种写法
不选中代码，表示在该位置新增代码(Generate)。如：提示输入一个数字n，打印“您输入的数字为：{n}”；若输入数字0，则退出///若输入的不是数字，加上错误处理

#### 9.2 Quick Question: Alt + Enter (在Ctrl + K之后的窗体显示之后才能操作)
针对选中的代码快速询问它的功能、用法、可能的改进方向
若同意，可以输入：do it
